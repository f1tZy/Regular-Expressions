## Описание:
Сайт с валидацией формы регистрации. Валидация для полей:
  - Имя
  - Email
  - Телефон
  - Сайт пользователя

## Стек: HTML, CSS.

## Инструкция как запустить:
* Скопировать репозиторий 
```sh
git clone https://github.com/f1tZy/Regular-Expressions.git
```
* Запустить файл index.html
```
